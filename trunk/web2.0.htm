    Adobe Flex ist ein Entwicklungsframework zum Erstellen von Rich Internet Applications (RIAs). Flex-Anwendungen haben den enormen Vorteil, dass sie nicht nur im Web, sondern auch auf dem Desktop laufen können und in allen gängigen Browsern gleich dargestellt werden.

    Wenn Sie für professionelle Rich Client Lösungen mit Flex interssieren, wir sind die richtige Partner!
    JavaFX ist eine Plattform, derer Entwicklungs- und Laufzeitumgebung auf Suns Java-Plattform basieren. JavaFX wurden für die Entwicklung von Rich Internet Applications entworfen. Auf die in Java enthaltenen Funktionen, Bibliotheken und Schnittstellen kann direkt zugegriffen werden.

    Mit JavaFX lassen sich plattformunabhängige Programme auch unter Windows und Linux erstellen und ist zukünftige Java Presentation Layer für plattformübergreifende Anwendungen.

    Eclipse Plugin und RCP
    Eclipse ist ein Open-Source Framework zur Entwicklung von Rich-Client-Applikationen, die bekannteste Verwendung ist die Nutzung als Entwicklungsumgebung (IDE), diese umfaßt unter der Bezeichnung Eclipse SDK die Eclipse Platform, Werkzeuge zur Java-Entwicklung (Java Development Tools JDT) und die Umgebung zur Entwicklung von Eclipse-Plugins (Plug-in Development Environment PDE).

    Wir haben mehrjährige Java-Programmierung Projekterfahrung, wenn Sie spezialle Java-Software, Eclipse-RCP, Eclipe-Plugin oder Java-Spiel entwicklen wollen, wir sind die richtige Parnter. Kontaktieren mit uns.
